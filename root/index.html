Documentation

Behat is a BDD tool for PHP 5.3. It uses Behat\Gherkin parser to parse plain text feature description files. Behat\Gherkin builds AST from your features and send them to Behat, that run testing definitions against them.

Behat was inspired by Ruby Cucumber project and shares same ideology. Most of the written Cucumber docs and articles can be applied to Behat.
